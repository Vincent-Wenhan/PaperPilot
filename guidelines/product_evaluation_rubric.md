# Product Evaluation Rubric

Score each dimension from 1 to 5:

- paper faithfulness,
- multi-paper coherence,
- user clarity,
- problem-solution fit,
- PRD completeness,
- MVP simplicity,
- demo feasibility,
- mock-first correctness,
- safety awareness,
- integration feasibility.

Below 4.0 overall requires revision suggestions. Never increase a score by
assuming unavailable models, datasets, checkpoints, or repository behavior.
